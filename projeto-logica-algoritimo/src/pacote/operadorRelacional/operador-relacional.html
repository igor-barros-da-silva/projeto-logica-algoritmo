Operadores Relacionais São utilizados para comparar String de caracteres, números, verificações, condições e tomada de decisão. Os valores a serem comparados podem ser caracteres ou variáveis. Estes operadores sempre retornam valores lógicos (verdadeiro/falso, true/false).
