O que é diagrama de bloco? É uma forma padronizada e eficaz para representar os passos lógicos de um determinado processamento. Com o diagram podemos definir uma sequência de símbolos, com significado bem definido, portanto, sua principal funçao é de facilitar a visualização dos passos de um processamento. ref: https://algoritmos10.webnode.pt/products/algoritmos-em-java/
