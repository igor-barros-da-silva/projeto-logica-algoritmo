Operadores Lógicos Os operadores lógicos servem para combinar resultados de expressões, retornando se o resultado final é verdadeiro ou falso. OU em Java | ou || OU em SQL OR E em Java & ou && E em SQL AND União C&C | C Quando usamos os operadores duplos (&& e ||) o Java não continua as verificações se o resultado já for conhecido Já usando os operadores simples (& e |), ambos os lados da operação são verificados.
