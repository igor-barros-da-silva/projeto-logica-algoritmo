Estrutura de repetição

São conhecidas como laços(loops) e são utilizados para executar, repetidamente, uma instrução ou bloco de intrução enquanto determinada condição estiver sendo satisfeita.

O que é sintaxe de código?

Em uma linguagem de progrmação o termo sintaxe refere-se ás regras da escrita


correta dos comando de texto para que os termos da linguagem executem e sejam interpretados e entendidos pelo computador

O que é semântica do código

É complementar à sintaxe de programas de computador, que se preocupa em descrever as estruturas de uma linguagem de programação. É escrever o código com a estrutura correta para a execução sequencial do código da linguagem e obtendo o resultado final.
